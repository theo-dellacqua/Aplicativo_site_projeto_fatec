Projeto Integrador VI

Big Data para Negócios • Machine Learning • Business Intelligence • Desenvolvimento Mobile

Sobre o Projeto

Este trabalho acadêmico, desenvolvido na disciplina de Projeto Integrador do curso de Big Data para Negócios da FATEC Ipiranga, tem como objetivo a análise de um caso real da empresa EGA Soluções Industriais, especializada no desenvolvimento de softwares para controle de chão de fábrica (MES).

O projeto utiliza técnicas de Big Data, Estatística, Inteligência Artificial, Modelagem de Dados, Machine Learning, Dashboards e Desenvolvimento Mobile, transformando dados industriais em informações estratégicas para apoio à tomada de decisão.

Sobre a Empresa

A EGA Soluções Industriais é uma empresa brasileira fundada em 1988 e atua há mais de três décadas no desenvolvimento de tecnologias para monitoramento e gestão da produção industrial.

Seu principal foco está nos sistemas MES (Manufacturing Execution System), que permitem acompanhar em tempo real o desempenho das operações produtivas, promovendo maior controle, eficiência e produtividade.

Entre as funcionalidades oferecidas estão o monitoramento de máquinas, cálculo de indicadores industriais, rastreabilidade de produção, gestão da qualidade, controle de inventário e acompanhamento do consumo de energia, contribuindo diretamente para iniciativas de Indústria 4.0.

Sobre o Aplicativo

Antes do desenvolvimento da aplicação Mobile, foi criado inicialmente um sistema Web utilizando as tecnologias Python e Django. Posteriormente, o projeto foi adaptado e transformado em aplicativo Mobile com auxílio do software Android Studio, permitindo maior mobilidade e acessibilidade aos usuários do sistema.

O aplicativo pode ser instalado em dispositivos móveis por meio de um arquivo gerado durante o processo de compilação da aplicação. Após a instalação, o usuário passa a ter acesso às funcionalidades desenvolvidas no projeto, incluindo dashboards, gráficos históricos e análises de Machine Learning.

A aplicação possui sistema de cadastro e autenticação de usuários. Durante o processo de cadastro, o usuário deve informar dados como nome de usuário, nome, sobrenome, e-mail, senha e confirmação de senha. Já para acessar o sistema, é necessário realizar login utilizando nome de usuário e senha cadastrados previamente.

O aplicativo está integrado ao banco de dados, permitindo atualização dinâmica das informações apresentadas nos dashboards e análises. Dessa forma, qualquer alteração realizada no banco de dados impacta diretamente os resultados analíticos disponibilizados pela aplicação.

Além disso, o sistema implementa mecanismos básicos de segurança e autenticação, garantindo que apenas usuários cadastrados e autenticados possam acessar os gráficos históricos e a análise de Machine Learning. Essa abordagem contribui para maior proteção das informações e controle de acesso aos dados industriais utilizados no projeto.
